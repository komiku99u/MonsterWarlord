# Monster Warlord Indonesia

Link Bio resmi komunitas Monster Warlord Indonesia.

## Features

* Responsive mobile-first design
* Lightweight and fast loading
* Cover image and profile avatar
* Download Game popup modal
* Social media links
* No external libraries
* No framework required

## Customization

### Profile Image

Replace:

```text
assets/avatar.jpg
```

### Cover Image

Replace:

```text
assets/cover.webp
```

### Social Links

Edit the links inside:

```html
index.html
```

### Download Sources

Edit:

```html
LINK_WEBSITE
LINK_PLAYSTORE
```

inside the download modal.
